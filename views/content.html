Hello im a an extension example, to have a good experience with me please follow the steps bellow:

content.html (./views/content.html): for editing DOM

extension.js (./js/extension.js): for editing js file, please respect the format given on the file

extension.css (./css/extension.css): for editing css properties

exemple-api-handler.js (./exemple-api-handler.js): file responsible of communication between extension and the API

index.js (./index.js): file responsible of loading, please note that if you rename the exemple-api-handler you must modify it also in index.js

manifest.json (./manifest.json): containing all properties of the extension such us css and js files path , author ...
